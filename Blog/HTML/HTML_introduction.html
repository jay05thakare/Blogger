Introduction

What is HTML?

HTML(Hypertext Markup Language) is used by developers to structure the webpage. Anywhere you navigate on any website, every single page is developed using new HTML file. HTML is language understood by the web-browser and instructs browser to display content. It is like a skeleton of a webpage. IT is a very easy language to understand and to implement. HTML tags for a HTML element, elements combine to form a part, parts combine to form a section, sections combine to form a webpage and webpages combine to form a website.
Even this page you are reading is made using HTML. This page is made of different sections. On-top you see a header navbar below to which is a sub-navigation bar and then comes the main section which is divided in vertical sections, and finally at bottom we have a footer. But HTML alone doesn't make a final product as it appears to us. As HTML acts as skeleton, CSS is needed to beautify the webpage which may be considered as muscles and skin of a human body, and Javascript is needed to add functions which may be considered as nervous system.
HTML has different components like tags and elements, attributes, meta data, etc.

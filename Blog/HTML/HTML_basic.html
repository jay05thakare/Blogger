HTML Basic

HTML has a basic structure code which helps browser to understand about the document.

Basic structure

HTML basic structure

 
<!DOCTYPE html>
<html>
<head>
  <title>Title</title>
</head>
<body>

  <h1>This is H1 heading</h1>
  <p>This is paragraph</p>
  <p>Hello World!</p>

</body>
</html>

  1. <!DOCTYPE html> defines that document is HTML5 file and contains HTML source code.
  2. <html> tag is base element of HTML page.
  3. <head> tag contains meta information of HTML page.
  4. <title> tag gives title to HTML page which is displayed in tab of browser.
  5. <body> tag contains structure of webpage. In this tag all the html code will be written.
  6. <h1> tag displays a H1 heading in browser.
  7. <p> tag displays a paragraph in browser.

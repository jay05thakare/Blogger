HTML Paragraph

<HTML paragraph> <p> tag is used to display context or to describe about heading . Content within <p> and </p> is displayed as paragraph.

<p> tag syntax

<p> paragraph_content </p>

HTML paragraph example

<p> Hello World! </p>

Paragraph tag properties

Paragraph tag doesn't give a sh*t about any indentation for a new line or any number of spaces except the first space. It just returns a string/sentence with a single space between two words. For example in code below there are many spaces in between words and some new lines. But still the output is a single sentence with single space betweeen words.

Example 1

<p> Dogs       are
    the best
                friend
anyone        can        have.
</p>

Note: To add extra space, we use &nsbp which is an HTML entity. We will learn more about entities in our HTML Entity chapter.

To change a paragraph, use another paragraph tag or you can add a <br> break tag to change the paragraph without shifting to a new <p> tag.

Example 2

<p>
  How are you?
</p>

<p>
  I am fine.
</p>

Using <br> tag to break sentences to new line.

<br> break tag bring sentence to new line. This is useful when you have to keep changing line, like while writting a poem or a conversation.

Example 3

<p> How are you? <br>I am fine. <p>

Note: You can use multiple <br> tags to get empty space after paragraph.

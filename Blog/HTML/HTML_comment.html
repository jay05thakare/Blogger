HTML Comments

In beginning, when code is simple and short, it is easy to remember which part of the code does what. But with time when code becomes complex, it's difficult to remember the work of each part of code. So HTML comment help a developer to understand which part of code performs which task. This makes work easy when changes need to be made. But HTML comment are not displayed on a browser. Browser while decoding HTML file, it skips HTML comment.

HTML comment tag

<!-- comment here --> This is comment tag
<!-- this is start tag of HTML comment tag and --> this is end tag of HTML tag. Content within these tags is considered as HTML comment.

HTML comment example

  <!-- start of section 1 -->
  <h1> This is section </h1>
  <p> This is section paragraph </p>
  <p> Observe the comment visible in browser. </p>
 <!-- end of section 1 -->

HTML Attribute

HTML attribute provide additional information about HTML element. HTML attribute is one of the important part of a webpage.

Attribute syntax

Attribute is placed in start tag of a element. Every html tag can have any number of attributes.
attribute_name = "attribute_value"
You'll understand it better in below examples.

Types of attribute

There are two types of attribute, 1.global attribute and 2.element specific attribute.

1. Global atrribute

Global attribute can be used on any of the element in a html file. Simply, whatever data you put in a global attribute, it a assigned to every element of a html file. Hence named GLOBAL attribute.

Global atrribute example
class = "class_name"
id = "id_name"
style = "property-1: value; property-2: value;"
title = "title_name"

1. Element specific atrribute

Element specific attribute can be used only on a specified element in a html file. Simply, whatever data you assign an element specific attribute is limited to that specific element only. Hence named ELEMENT SPECIFIC attribute.

Element specific atrribute example
1. Element specific atrribute href = "url"
src = "url/path"
width = "value"
alt = "alternate_text"

href attribute

The <a> tag defines hyperlink. The <href> attribute specifies the url of the page link goes to:

HTML href attribute example

<a href=" http://www.amazon.com "> Amazon </a>

You will learn more about links in our HTML Links chapter.

src and alt attribute

The <img> tag is used to embed an image in a HTML page. The src attribute specifies path/url to the image to be displayed.
The required alt attribute for the <img> tag specifies an alternate text for an image, if the image for some reason cannot be displayed. This can be due to slow connection, or an error in the src attribute, or if the user uses a screen reader.

HTML img attribute example

<img src="  image_url/path.jpg  " alt="  alternate_text  >

You will learn more about images in our HTML Images chapter.

Width & height attribute

The <img> tag should also contain the width and height attributes, which defines width and height of image respectively in pixels.

<img src=" image_url/path.jpg "   width="100px"   height="100px">

title attribute

The <title> attribute defines some extra information or value about an element when cursor is over an element.

HTML title attribute example

<button  title=" Submit "   > Save </button>

style attribute

The <style> attribute is used to add styles to an element, such as color, font, weight, size, and more.

HTML title attribute example

<p  style=" color: yellow "   > This is yellow paragraph. </p>

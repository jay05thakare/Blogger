HTML Heading

HTML headings are titles or subtitles of a webpage, which are important in terms of search engines. We use <h> tag to display heading.

Types of HTML heading

There are six types of heading from <h1> to <h6>, having a different font size and weight with descending priority level. H1 heading being most important to H6 being least. Every heading has some default font, size, and weight, but they can be changed using styles or CSS.

Syntax:
<h1> heading </h1>

Title and targeted keywords should be present in the H1 heading of a page, it's not mandatory but best practice for SEO.

Note: Do not use heading tag to describe paragraph to obtain bold and big content. Use heading tag for headings only.

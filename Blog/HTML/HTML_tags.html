HTML Tags

HTML tags are hidden keyword in a page that defines how a browser must display/structure the content.

Types of tags

There are two types tags, 1.Paired tag and 2.Single tag.
Paired tag has opening and closing tag like <p> </p>.
Single tag has only opening tag like <img>.

Content within tag is collectively called HTML element.

Tags categories

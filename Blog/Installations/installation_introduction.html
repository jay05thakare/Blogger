Tools required to start coding.

Installing a environment offline for writing code makes it very handy to start anywhere anytime.

A very basic tool, a device which maybe a smart phone, a tablet, laptop or desktop. Smart phone (also some android tablets) does not need/support a installed environment. You can install Decoder Mobile App to compile code online. But a installed environment is best for other devices.

Tools required to get started with coding

  1. IDE
  2. Compiler
  3. Web browser

IDE

Very first thing you need to get to start is a text editor, a place where you will type or write code.

IDE(Integrated Development Environment) is a software application which provides an evironment with specially designed text editor. IDE normally consist at least a source code editor, build automation and a debugger. There are many IDEs like VS Code, NETBeans, IntelliJ, etc. But IDE developed by microsoft and used by many professional, VS Code (Virtual Studio Code) is the best code editor.

(P.S- You should know that IDE does not compile or gives output. Compiler does this work.)

Downloading and installing VS Code

Compiler

A compiler is a special program that processes code written in a particular programming language and turns them into a machine language or code that computer's processor understands. There are hindreds of programming languages, therefore it's sensible to download only that compiler which you need for a programming language.

(P.S- You dont need a compiler for web-development, i.e for markup language HTML CSS. You need a web browser in this case.)

Web browser

Web browser like google chrome, mozilla firefox, etc is required when you are dealing with markup language

(P.S- You dont need a compiler for web-development, i.e for markup language.)

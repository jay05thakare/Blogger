Virtual Studio Code Download and Installation

Virtual Studio Code is an IDE developed by microsoft and chose by most Programmers. VS Code provides environment to write and execute code.

Why choose VS Code

  1. 1. Lightning fast source code editor.
  2. 2. User friendly interface.
  3. 3. Super productive with syntax highlightning, box-selection, auto indentation, bracket-matching snippets, etc
  4. 4. Keyboard shortcuts and easy customization
  5. 5. In-built git support

  1. Downloading VS Code setup file

    1. 1. Google search "Download VS Code" and go to code.visualstudio.com > download
    2. 2. Choose your operating system from Windows, Linux and IOS, and downloading of a .exe file will start automatically.

  2. Run VS Code setup wizard

    VS Code setup file is downloaded sucessfully. Run the setup file from location of download.

    1. 1. Setup - VS Code

      At first it recommends to close all windows but its not mandatory.

    2. 2. License Agreement - VS Code

      This is an agreement and you know how to read it in a second. It's safe so click on 'I agree' check-box.

      You cannot go ahead till you accept the agreement.

    3. 3. Select destination location - VS Code

      Choose location in your system, but make sure the location is in local disk c where your operating system and other software are stored. If you don't have any specific reason to change location, then it's best to go with default location.

      There should be 293.2 MB free disk space.

    4. 4. Select start menu folder - VS Code

      As a default, VS Code shortcut will be added in 'Start' menu folder. We can change destination or skip creating shortcut.

    5. 5. Select additional tasks - VS Code

      One of the important step! We get this important additional tasks. These tasks are handy so check all boxes unless you have any specific reason to uncheck them.

    6. 6. Ready to install - VS Code

      Everything is properly set and VS Code is ready to get installed in system.

    7. 7. Finish and launch - VS Code

      After succesful installation, we can launch VS Code in our system.

      You have to restart your device before using it for the first time. This step is mandatory.

Congratulation! We have successfully downloaded and installed Virtual Studio Code IDE.
